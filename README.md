# Robotics Education Projects

A collection of educational robotics projects, sample code, and selected course materials used in my robotics and embedded systems classes.

This repository is **not intended to be a complete robotics course**. Instead, it serves as a companion resource for students, providing reference implementations, selected examples, and supporting materials discussed during the classes.

The complete learning experience—including lectures, engineering discussions, project design, debugging sessions, and hands-on mentoring—is delivered during the classroom sessions.

---

## Educational Philosophy

My goal is not simply to teach students how to assemble robots.

The focus is on developing engineering thinking through real-world projects that combine software, electronics, sensors, mechanics, and system design.

Students progressively learn how to:

* Design embedded systems
* Read and understand hardware datasheets
* Program microcontrollers in C/C++
* Integrate sensors and actuators
* Debug hardware and software
* Analyze engineering problems
* Build reliable robotic systems

Rather than following step-by-step tutorials, students are encouraged to understand *why* a system works before implementing it.

---

## About This Repository

This repository contains only a subset of the materials used during the courses, including:

* Sample Arduino projects
* Reference implementations
* Selected lecture notes
* Supporting Python exercises
* Mechanical models used in class

Many advanced projects, engineering discussions, design decisions, and practical workshops are intentionally reserved for the classroom environment, where students receive direct guidance, feedback, and mentoring throughout the development process.

## Repository Structure

### arduino-codes/

Reference examples demonstrating fundamental embedded programming concepts, including GPIO, sensors, displays, motors, and simple robotic behaviors.

### class-notes/

Selected lecture notes and reference documents used during the courses.

### python-course/

Preparatory Python exercises for students before entering embedded programming and robotics.

### solidworks-models/

Mechanical models and educational CAD files used in selected classroom projects.

---

## Learning Beyond the Repository

Engineering is learned by building, experimenting, making mistakes, and discussing solutions.

While this repository provides useful reference material, the complete educational experience comes from actively participating in the projects, laboratory sessions, and engineering challenges developed during the courses.
